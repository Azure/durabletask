# Sample Applications

This page provides an overview of the sample applications included in the repository. These samples demonstrate various patterns and features of the Durable Task Framework.

For detailed instructions on running each sample, see the README in each project directory.

## Sample Projects

| Sample | Description | Key Features |
| ------ | ----------- | ------------ |
| [DurableTask.Samples](../../samples/DurableTask.Samples/) | Core orchestration patterns | Greetings, Cron, Fan-out/Fan-in, Error handling, External events |
| [Correlation.Samples](../../samples/Correlation.Samples/) | Legacy distributed tracing | W3C TraceContext, Application Insights |
| [DistributedTraceSample](../../samples/DistributedTraceSample/) | Modern telemetry integration | OpenTelemetry, Application Insights |
| [ManagedIdentitySample](../../samples/ManagedIdentitySample/) | Azure authentication | Managed Identity with Azure Storage |
